Atualização entregue nesta versão

1. Dashboard com cards visuais em imagem
- A tela inicial foi redesenhada com cards grandes para Próximo Culto, Próximo Evento, Membros, Escalas e Manutenções.
- As imagens foram adicionadas localmente em assets/cards/ para funcionar no GitHub Pages e no PWA.

2. Dark Mode / Light Mode mantido
- O botão de alternar tema continua no canto superior.
- O tema claro foi mantido de forma completa, incluindo menu lateral e barra inferior no celular.

3. Financeiro e Doações
- Foram adicionadas as telas Financeiro e Doações.
- Os dados também sincronizam pelo Firebase/Firestore.
- Essas telas exigem que o perfil tenha senha cadastrada.

4. Usuário com senha
- Ao criar um novo perfil, agora é obrigatório cadastrar uma senha.
- Perfis antigos sem senha continuam entrando, mas precisam definir senha para acessar Financeiro e Doações.

5. Editar usuário
- Foi criado botão de edição do usuário no rodapé do menu lateral.
- Também existe o botão em Configurações > Meu perfil.
- É possível alterar nome, ministério, cargo, foto e senha.

Observação de segurança
- A senha implementada protege o uso dentro do aplicativo, mas como o app roda no navegador e o Firestore está em modo de teste, isso não substitui regras de segurança definitivas do Firebase.
- Antes de usar dados financeiros reais, configure regras mais seguras no Firebase.


Atualização 20260704-cachefix-v7
- Corrigido problema de cache/PWA que fazia o navegador continuar mostrando versões antigas.
- index.html agora chama style.css, script.js e manifest.json com versão na URL.
- service-worker.js agora usa network-first para HTML/CSS/JS/JSON e cache-first só para imagens/ícones.
- O app tenta aplicar novas versões automaticamente sem exigir limpeza manual de cache.
